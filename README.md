# Dashboard Comercial — datacrazy

Dashboard web para gestão do pipeline comercial.

## Pasta

Tudo fica em `C:\Users\naahl\dashboard-datacrazy`.

## Setup inicial (uma vez só)

Abre o Prompt de Comando, navega até a pasta e roda:

```cmd
cd C:\Users\naahl\dashboard-datacrazy
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Testar conexão

Ainda com o ambiente ativado (`venv\Scripts\activate`), roda:

```cmd
python test_conexao.py
```

Se aparecer a lista de pipelines no final, conexão OK.
